# youtube-shorts-wen-scraping
A simple and efficient Python script to extract metadata from YouTube Shorts without an API key with an automatic JSON export .

The tool automatically extracts a set of metadata : 

* url : The original Shorts link

* video_id : The video’s unique identifier

* title : The full title of the video

* description : The complete description exactly as it appears on YouTube

* channel : The name of the channel that uploaded the video

* channel_id : The channel’s unique ID

* publish_date : The raw publication date in YYYYMMDD format

* duration_seconds : The video’s length in seconds

* views : Total number of views

* likes : Total number of likes

comments_count : Number of comments posted

thumbnail : Direct link to the video’s thumbnail

formatted_date – A clean, human-friendly publication date
